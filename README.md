# Three-Black-Queens
AI Chess Engine Submission for DSU's AI Chess Competition!

## Description
This is a improved version of the minimax algorithm implementation in the form of a chess engine. It was created within 2 weeks as a submission by 3 undergraduates at DSU majoring in Artificial Intelligence. 

## Teammates
The team included Connor Brenner (https://github.com/cnnrbrnnr), Ethan Savage (https://github.com/Ethan-Savage), and Sureh San (https://github.com/SurehSan). 

## Lessons Learned
Throughout the 2 weeks of the competition, we improved in our soft skills aswell as our technical skills:
- Python techniques such as utilizing new libraries (chess, chess.pyglot, Interface, etc)
- Documentation reading aswell as creating clean documentation
- Teamwork, communication, and collaboration
- Algorithmic Skills 
- Learning and applying chess logic to tangible code
